# World Development Archetypes and Analytics Framework
▪ End-to-end statistical analysis of global development indicators in R, including missing-data assessment, MICE-based multivariate imputation, correlation analysis, and exploratory visualization across socioeconomic, health, and trade datasets.

▪ Principal Component Analysis (PCA), hierarchical clustering, and K-means clustering to identify latent development patterns across countries, reducing dimensionality and profiling global socioeconomic clusters through interpretable visual analytics and heatmaps.

▪ Reproducible data science workflow using tidyverse, ggplot2, corrplot, and clustering libraries to evaluate development trends, validate cluster structures, and generate publication-style analytical visualizations for multivariate country-level analysis.
